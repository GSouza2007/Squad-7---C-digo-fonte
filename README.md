# CotaWeb Onboard

Interface interativa para onboarding de usuários na plataforma CotaWeb.

## Tecnologias

- HTML5
- CSS3 (Flexbox, Responsive Design)
- JavaScript Vanilla

## Estrutura

- `index.html` - Interface principal
- `styles.css` - Estilos e responsividade
- `script.js` - Lógica de interação

## Funcionalidades

**Passo 1 - Novo Fornecedor**
- Formulário com campos obrigatórios
- Formatação automática de telefone e CNPJ

**Passo 2 - Cadastre seus produtos**
- Adicionar produtos com modal
- Tabela dinâmica de produtos

**Passo 3 - Crie uma cotação**
- Seleção de produtos
- Visualização da cotação

**Passo 4 - Envie sua cotação**
- Seleção de fornecedores
- Envio de cotações

## Recursos

- Indicador de progresso com 4 etapas
- Accordion expansível para cada passo
- Validação de campos obrigatórios
- Design responsivo
- Transições suaves

## Como usar

1. Abra `index.html` no navegador
2. Complete cada etapa do tutorial
3. Clique em "Salvar" para avançar
4. Finalize após completar 3 de 4 passos
